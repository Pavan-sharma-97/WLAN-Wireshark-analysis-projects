# 🚀 QoS Data Frame (Subtype 1000)

## Purpose
Enhanced data frame supporting:
- EDCA
- TXOP
- TID prioritization
- Block ACK
- A-MPDU aggregation

## Additions
Includes **QoS Control Field**:
- TID (bits 0–3)
- EOSP
- ACK Policy
- Queue Size / Buffer state

## Used by
All 802.11e/n/ac/ax devices
(WMM Certified)
