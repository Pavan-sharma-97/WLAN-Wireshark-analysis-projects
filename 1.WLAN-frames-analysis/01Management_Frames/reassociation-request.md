# 🔄 Reassociation Request (Subtype 2)

Used when STA roams between APs within the same ESS.

---

## Addresses
| Address | Meaning |
|--------|---------|
| Address1 | Target AP |
| Address2 | STA |
| Address3 | Current AP BSSID |

---

## Additional Field
- Current AP Address (the AP STA was associated with before)
